#const 关键字

const关键字修饰的所有变量会保存在存储区区域，同时在获取地址时会在栈上分配一个空间

## 基本数据类型 ##

``````cpp
//存储区存储
const int data=1;
//在栈上分配内存返回地址也是栈上的
int *ptr2=(int*)&data;
//修改的地址是栈上的地址
 *ptr2=10;
  cout<<"data="<<data<<"*p="<<*ptr2<<endl;
//data=1*p=10 存储区区域的数据不变
``````

## 引用数据类型 ##

引用数据类型和基本数据类型差距比较大

``````cpp
//数据存储在堆上，存储区区域存储地址  
const Obj obj={"小红",12};
//读取到的地址是堆上的
    Obj *p=(Obj*)&obj;
//直接修改堆上的数据
 p->name="小安";
 cout<<obj.name<<endl;
``````
